# 管理端功能重构清单

原则
- 不删除现有任务、员工、请假、出差、导入导出、识别功能。
- 以管理者视角做收敛，而不是再铺一层新系统。
- 优先做跨模块统一入口、异常视图和员工主线。

P0
- 管理者待关注面板
  - 统一展示逾期任务、请假占用、出差异常。
  - 作为首页管理入口，不替代原有列表页。
- 员工主线统一
  - 员工详情可串起任务、请假、出差。
  - 保留原团队页、考勤页、任务页。
- 异常视图
  - 出差缺勤未填原因。
  - 出差补贴超周期未备注。
  - 逾期未完成任务。
- 统一筛选范式
  - 搜索、状态、部门、周期的交互一致。

P1
- 员工详情页或侧滑面板
  - 统一查看个人任务、请假、出差。
- 操作留痕
  - 记录创建、编辑、删除、导入。
- 统一导出入口
  - 从模块导出提升到管理报表入口。

P2
- 常用筛选保存
- 经理视角报表
- 首页搜索联动
