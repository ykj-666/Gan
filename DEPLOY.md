# Gan 部署

项目只保留一个部署入口：

```bash
node deploy.cjs
```

Windows 下也可以直接双击 `start.bat`。

脚本现在只做线上部署这件事：

1. 检查 Node.js 版本
2. 准备 `ssh2` 依赖和本机 SSH 密钥
3. 检查服务器免密登录，必要时补配公钥
4. 打包项目代码
5. 上传部署包和生产环境变量
6. 在服务器安装依赖、构建、迁移 MySQL、启动 PM2

- 如果 `3000` 端口无法访问，先检查云服务器安全组
