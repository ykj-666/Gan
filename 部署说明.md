# Gan 部署说明

现在只有一种部署方式，不再区分网页部署、命令行菜单部署、本地开发启动。

## 使用方法

在项目根目录执行：

```bash
node deploy.cjs
```

Windows 可以直接双击：

```text
start.bat
```

## 脚本会自动完成

1. 检查本机 Node.js 版本是否满足要求
2. 安装部署依赖 `ssh2`
3. 生成本机 SSH 密钥
4. 首次部署时把公钥写入服务器
5. 打包项目代码
6. 上传部署包和生产环境变量
7. 在服务器安装依赖并构建
8. 初始化 MySQL
9. 用 PM2 启动服务

## 重要提示

- 生产环境必须配置 MySQL `DATABASE_URL`
- 首次部署如果提示输入 `ubuntu` 密码，这是为了配置 SSH 免密登录
- 后续再部署一般不需要重复输入密码
- 如果页面打不开，先检查安全组是否放行 `22` 和 `3000`
