# 🚀 Gan 项目部署说明（纯小白版）

## 一、准备工作（只做一次）

### 1. 安装 Node.js

如果您的电脑还没有安装 Node.js：

1. 打开浏览器访问：https://nodejs.org/dist/v20.18.0/node-v20.18.0-x64.msi
2. 下载后双击安装，全部点击"下一步"即可
3. 安装完成后重启电脑

### 2. 确认项目代码在本地

确保您已经把代码下载/克隆到本地文件夹，比如 `D:\Gan`

---

## 二、一键部署到服务器

### 方法一：双击运行（推荐）

1. 打开项目文件夹
2. **双击 `点击部署.bat`**
3. 会弹出一个蓝色窗口，显示中文菜单

### 方法二：命令行运行

1. 在项目文件夹的地址栏输入 `cmd`，按回车
2. 输入命令：`node deploy.js`
3. 回车运行

---

## 三、操作说明

运行后会看到菜单：

```
╔══════════════════════════════════════════╗
║       Gan 项目一键部署工具               ║
║       服务器: 118.89.134.207            ║
╚══════════════════════════════════════════╝

  [1] 一键部署到服务器（首次或更新）
  [2] 启动本地开发服务器（本地测试）
  [3] 查看服务器运行状态
  [4] 查看部署说明
  [5] 退出
```

### 首次部署（选择 1）

1. 按 `1`，回车
2. 如果是第一次，会提示输入 **服务器 root 密码**
3. 输入密码（输入时不会显示，这是正常的），回车
4. 等待自动完成（大约 2-5 分钟）
5. 看到绿色 "部署成功" 提示即可

### 更新代码（选择 1）

代码修改后：
1. 双击 `点击部署.bat`
2. 按 `1`，回车
3. **不需要再输入密码**，自动完成更新

### 本地测试（选择 2）

不想部署到服务器，只想在本地测试：
1. 按 `2`，回车
2. 自动启动本地服务器
3. 浏览器打开 http://localhost:3000
4. 关闭黑色窗口即可停止

## 五、常见问题

### ❓ 部署后打不开网页

**原因：** 腾讯云安全组没有开放 3000 端口

**解决方法：**
1. 打开腾讯云控制台
2. 找到您的服务器 → 安全组
3. 点击「入站规则」→「添加规则」
4. 填写：协议 TCP，端口 3000，来源 0.0.0.0/0
5. 保存，稍等几分钟再试

### ❓ 提示 "未检测到 Node.js"

**原因：** 电脑没有安装 Node.js

**解决方法：** 看上面的「一、准备工作」

### ❓ 提示 "SSH 连接失败"

**原因：** 服务器 22 端口未开放，或密码错误

**解决方法：**
1. 确认服务器 IP 是 `118.89.134.207`
2. 确认 root 密码正确
3. 在腾讯云安全组中确认开放了 TCP 22 端口

### ❓ 更新后数据会丢失吗？

**不会。** 数据库是 SQLite 文件（`local.db`），每次更新时会自动备份和恢复。
